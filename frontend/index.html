How to use Roamable
Map controls
🔍
Search barType a place or pub/café name. Tap a result then "↗ Route here" to navigate there from your location.
⊕
Find meCentres the map on your current GPS location.
🔄
Rotate mapTwo-finger twist to rotate. Tap the N button to snap back to north-up.
Markers
＋
Drop a markerTap ＋ to drop a marker at your current location. Use the 📍 pin button to tap anywhere on the map instead.
👆
View a markerTap any marker to open its detail panel — photo, difficulty, and community updates.
✏️
Edit a markerIn the marker panel, tap ✏️ Edit to change type, difficulty or note.
💬
Add an updateIn the marker panel, type a comment (e.g. "gate removed") and tap Post.
Routes
↗
Plan a routeTap the route button, then tap start and end points. Tap Done when finished.
🚧
Avoid obstaclesAfter a route is drawn, use the avoid panel to route around hard or tricky markers.
Coverage (☰ menu)
◉
Explored pathsShows all walks you've imported as a purple coverage band, plus total distance explored.
📂
Import GPXImport a recorded walk. Only new ground is added — already-covered sections are skipped.
📥
Save offlineCaches the current map view for use without internet. Turn on coverage layer first.
Install as an app
📱
Add to home screeniPhone: Share → Add to Home Screen. Android: menu → Install app.
